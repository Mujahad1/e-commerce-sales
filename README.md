# E-Commerce Sales Analysis

## Business Problem
An e-commerce dataset with 102 orders was analysed to identify 
revenue leakages, fulfilment issues, and payment risks.

## Key Findings
- Only 10.8% of orders were successfully delivered
- 47% of revenue (₹63,421) is at risk from cancellations and returns
- Cash on Delivery has a 21.2% cancellation rate vs 8.7% for Credit Card
- Clothing has the highest average order value (₹2,466) despite not 
  being the top revenue category
